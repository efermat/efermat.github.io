Fast Data Acquisition and Real-Time Processing Techniques for Through-the-Wall Radar Imaging

Authors: Wenji Zhang and Ahmad Hoorfar

Source: FERMAT, Volume 3, Article 2, May_Jun, 2014

Abstract: One of the main challenges in through-the-wall radar imaging (TWRI) is the long data acquisition and/or data processing time. The movement of the target during the long data collection time will cause smearing and target displacement in the through-the-wall image. Rapid data processing is critical to the real-time monitoring of targets behind the wall. In this paper, we present three techniques for fast data acquisition and real-time processing for TWRI. The first technique is based on compressive sensing incorporating the layered media Greenâ€™s function and is capable of dealing with targets behind multilayered building walls. Next, we formulate the beamforming algorithm for through-the-wall imaging with multiple-input multiple-output (MIMO) radar which is real-time in data collection. Finally, we propose a real-time processing algorithm for radar imaging through single or multilayered walls. Numerical results are presented to show the efficiency and the promising applications of the above techniques for fast data acquisition and real-time processing in TWRI.

Index Terms: TWRI, fast data acquisition, real-time processing, compressive sensing, MIMO radar, diffraction tomography.
